网页版俄罗斯方块游戏

===============

一个使用HTML5 Canvas和JavaScript实现的经典俄罗斯方块游戏。

功能特点：
- 七种经典俄罗斯方块形状
- 完整的游戏逻辑（移动、旋转、消除、计分）
- 现代化UI设计，响应式布局
- 音效和视觉特效
- 本地最高分记录
- 难度等级选择
- 深色/浅色主题切换

操作方法：
- 左右箭头键：移动方块
- 上箭头键：旋转方块
- 下箭头键：加速下落
- 空格键：开始/暂停游戏
- R键：重新开始游戏
- P键：暂停/继续游戏

运行方法：
1. 双击打开 index.html 文件
2. 或者使用任何现代浏览器打开该文件

游戏设置：
- 可以在游戏界面右侧调整难度、音效和主题

技术栈：
- HTML5 Canvas
- CSS3
- JavaScript (ES6+)
- 本地存储 (localStorage)

文件结构：
- index.html: 主页面
- style.css: 样式文件
- game.js: 游戏逻辑
- README.txt: 说明文件
